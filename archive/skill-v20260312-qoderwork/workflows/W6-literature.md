# W6: 文献研究（学术/理论深度调研）

**目标**：系统性了解某领域的学术共识和证据
**典型耗时**：L3 = 2-4h

## 步骤

### SCOPE
研究问题（用 PICO 或类似框架精确定义）+ 完成标准。

### ORIENT（建立领域地图）
- WebSearch "site:semanticscholar.org": 关键词 → Top cited papers
- Connected Papers: 以 1 篇核心 paper 为起点展开引用网络
- 社区综述: "X survey" / "X review" 找已有综述

### SELECT
选 2-3 个子领域重点深入。

### DEPTH（精读 + 声明-来源配对）
- 每个关键发现附 [论文标题, 年份, 来源URL]
- 每 5-7 篇写 3 句中间综合
- 注意证据质量层级：RCT > 准实验 > 观察性 > 专家意见

### CHALLENGE（对抗性检验）
- 搜索 "[结论] criticism / limitation"
- 至少找 1 个挑战性来源
- 标注争议点和未定论的问题

### SYNTHESIZE
回答研究问题，标注确信度：确信 / 可能 / 不确定。

### VERIFY（仅 L3）
关键声明溯源表：

| 声明 | 来源 | 原始文本 | 验证状态 |
|------|------|---------|---------|

### PERSIST
按持久化规则写报告 + 更新 INDEX.md。
