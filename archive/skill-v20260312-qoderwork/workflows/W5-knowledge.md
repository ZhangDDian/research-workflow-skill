# W5: 知识扩展（"最新技术/技巧"）

**目标**：扩展对某领域的认知，获取最新信息
**典型耗时**：L2 = 15-30min

## 步骤

### SCOPE
当前认知边界 + 想扩展的方向。

### BFS SCAN（快速扫描最新动态，并行）
- 实时: Tavily "X latest" / "X 2025-2026"
- 社区: HN "X" / Reddit "r/X" 最新热帖
- 技术: arXiv（如果是学术前沿）/ GitHub trending
- 中文: 掘金/知乎最新回答

### CURATE
筛选高价值信息（≤10 条），每条附来源。

### CONNECT
与已有认知的关联：
- 这改变了什么之前的假设？
- 这打开了什么新的可能性？
- 这有什么需要注意的坑？

### PERSIST
按持久化规则写报告 + 更新 INDEX.md。
时效性内容标注日期。
