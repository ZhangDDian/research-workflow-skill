# W4: 最佳实践调研（"别人怎么做的"）

**目标**：找到领域内最佳实践和成功案例
**典型耗时**：L2 = 20-40min, L3 = 1-2h

## 步骤

### SCOPE
明确"最佳"的评判标准：规模？效果？成本？适配性？

### SURVEY（广度搜索案例，并行）
- 英文: Exa/Tavily "best practices for X" / "how [top company] does X"
- 中文: 知乎 "X 最佳实践" / 36kr / 公众号
- 学术: "X case study" / "X framework"

### RANK
按评判标准排序 Top 5 案例。

### DEEP DIVE
选 Top 2-3 深入分析：
- 具体做法拆解
- 适用前提和约束条件
- 可迁移的部分 vs 不可迁移的部分

### ADAPT
结合自身场景的适配建议：
"他们做了 X，我们的约束是 Y，所以建议改为 Z"

### PERSIST
按持久化规则写报告 + 更新 INDEX.md。
